Experimental Environment

We ran the experiments on a Sun Grid Engine cluster with 24 nodes running Rocks Linux with 16 GB RAM and 2 quad-core Intel(R) Xeon(R) CPU E5410 CPUs @2.33GH processors each. Every single program was run non-exclusively on one core.

You can also run the experiments on one machine, though, see below for scripts.
